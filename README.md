# 🎬 Movie Search App

A movie search application built with HTML, CSS and JavaScript using the OMDb API.

## ✨ Features

- 🔍 Search movies by title
- 🎬 Display movie posters
- ⭐ View movie details
- 🚫 Fallback image for missing posters
- ⚡ Fast and responsive interface


## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- OMDb API
